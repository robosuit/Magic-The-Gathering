## How to use this folder
1. Copy one of the existing deck notes into `Cards/Friends Libraries/Calebs Stuff/Decks/` (right-click → "Duplicate").
2. Rename the copy to your desired deck name.
3. Use **Deck Template** to create a new deck from scratch.
4. Build your deck by checking cards off in **Card Library**.
