Deployment Notes

What still happens on GitHub

The local scaffold is ready. Repository settings still require your authenticated GitHub account.

  1. No folder move is needed. This repository already sits at the correct root.
  2. Push the new root files and the assets/ folder to your default branch.
  3. In GitHub, open Settings -> Pages, choose Deploy from a branch, then select main and / (root).
  4. Keep the source repository private if you want, but plan around the published GitHub Pages site being public.
