# Jund Token Swarm Deck (Abzan Variant)

## 🧬 Deck Information
- **Format:** 60-65 Card Casual
- **Focus Colors:** Green / White / Black (Abzan)
- **Strategy:** Token generation + Lifelink + Swarming overrun
- **Budget:** $60-75 complete
- **Status:** In Development (Version 1)

---

## 📊 Strategy Overview

### Early Game (Turns 1-3)
- Ramp with mana creatures
- Begin generating small tokens
- Stabilize with cheap blockers

### Mid Game (Turns 3-5)
- Swarm board with multiple tokens
- Add lifelink buffs for defense & offense
- Control via board presence

### Late Game (Turn 6+)
- Deploy massive finishers
- Overrun opponent with board advantage
- Use lifelink to stabilize life total

## 🎯 Core Synergies

### 1. Token Multiplication
- Imperious Perfect generates 1/1 elves
- Tokens can attack or block
- Sacrificial outlets for value

### 2. Lifelink Layer
- Gain life while attacking
- Stay ahead on life total
- Survive aggressive decks

### 3. Big Finisher Layer
- Token swarm softens opponent
- Deploy Polukranos or Deus for lethal
- Trample + lifelink = unstoppable

## 🚀 Play Pattern

**Example Turn 4-6 sequence:**

1. Turn 1-2: Play ramp + token generator
2. Turn 3: Attack with tokens, hold back blockers
3. Turn 4: Play anthem/buff, add more tokens
4. Turn 5: Let opponent see 8+ creatures, add lifelink
5. Turn 6: Deploy finisher + overrun

This pattern keeps pressure while staying safe.
