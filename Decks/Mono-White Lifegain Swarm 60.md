## Game Plan
1. Start on cheap creatures and early life gain so [[Ajani's Pridemate]] becomes a real threat instead of a filler two-drop.
2. Use [[Sol Ring]] or [[Heraldic Banner]] to jump ahead on mana while still supporting the go-wide plan.
3. Use [[Resolute Reinforcements]], [[Release the Dogs]], [[Felidar Retreat]], and [[Regal Caracal]] to build a wider board than spot removal can answer cleanly.
4. Close the game with [[Linden, the Steadfast Queen]], [[Syr Alin, the Lion's Claw]], [[Lyra Dawnbringer]], or an aura-backed attacker.

## Flex Slots
- [[Angel of Finality]] can move into the main deck if graveyard decks are common.
- [[Fumigate]] is the first swap in when the table gets wider or faster than this list.
- [[Basilisk Collar]] is usable here, but it is not as clean a main-deck upgrade as [[Sol Ring]] or [[Heraldic Banner]].
- If you add more copies of [[Ajani's Pridemate]], [[Inspiring Overseer]], or [[Regal Caracal]], replace the weakest one-of creatures first.
